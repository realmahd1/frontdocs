**۲. مشکل در ارتباط بین بخش‌های سیستم است؟**

مثلاً همه‌جای پروژه مستقیماً به یک API خارجی وابسته شده است:

```
axios.post(
  "external-api.com/send",
  data
);
```

اینجا یک Adapter می‌تواند ارتباط را ساده کند:

```swift
externalService.send(data);
```
